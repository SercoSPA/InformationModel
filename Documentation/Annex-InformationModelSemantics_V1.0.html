Information Model Semantics

Following the WC3 OWL reference standard, the semantics of an information model is implemented by the following logical entities:

  • Individuals (classes or sub-classes), providing an abstraction mechanism for grouping elements with similar characteristics
  • Properties, providing an instance of individuals, in particular:
    • Object properties, linking individuals to other individuals
    • Data properties, linking individuals to data values
